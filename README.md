## Installation ğŸ’»
1. **Clone the repository**

    ```bash
    git clone https://github.com/ryusudol/VitaAI.git
    ```

    ```bash

    ```
    * recommended npm version: 10.8.0

    ```bash
    OPENAI_API_KEY="Your OpenAI API Key"
    ```

4. **Start the project**

    ```bash
    npm run dev
    ```

    ```bash
    npm start
    ```
